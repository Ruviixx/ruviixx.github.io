En este proyecto analizaremos la medidas de seguridad de nuestra propia BIOS.

A continuacion analizaremos las diferentes medidas de seguridad que tiene nuestra UEFI

Contraseña de administracion

Si se activa nos solicitara una contraseña para modificar los parametros de la BIOS.

Contraseña de usuario de la BIOS

Si la activamos nos solicitara una constrañara para modificar los parametros basicos de la BIOS.

Permiso para el arranque desde dispositivo USB

Dependiendo si esta habilitado o no se podra arrancar el ordenador desde unidades extraibles a traves del puerto USB

Secure Boot

Introducido por Microsoft y cuya finalidad es denegar la ejecucion en el arranque de software no firmado que no sea Windows.

Orden de arranque

En esta opcion podremos elegir el orden de preferencia que la BIOS va a utilizar para arrancar el pc
